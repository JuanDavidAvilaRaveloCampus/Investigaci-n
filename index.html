CSS Flexbox Layout Module

Before the Flexbox Layout module, there were four loyaut modes

  • Block, for sections in a webpage
  • Inline, for text
  • Table, for two-dimensional table data
  • Positioned, for explicit position of an element

The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

PROPIEDADES

Flex-direction:

Permite manupular la forma en la que se va a mostrar los elementos apilados dentro de nuestro contenedor de flex.

  • row:

Valor por defecto. Apila nuestros elementos como si fuera una fila

  • Column:

Aplica estilos en forma de columnas.

  • Flow:

Permite unir las características de dos propiedades. Como por ejemplo "flex-direction: Flow: column grap;"

  • -reverse:

Invierte el orden de los items de cualquier propiedad.

  • wrap:

Permite a los elementos dar un salto de linea y ubicarse justo debajo de otro elemento creando una nueva fila para no alterar el tamaño de los items.

no-wrap

Sin wrap. Usualmente este valor viene por defecto.

  • Flex-start:

Alínea elementos a la izquierda.

  • Flex-end:

Alínea elementos al a derecha

  • Center:

Centra los elementos horizontalmente

  • Space-around:

Agrega un espacio alrededor de los elementos

  • Space-between

Agrega un espacio entre los elementos, mas sin embargo no entre el contenedor del elemento.

  • Space-evely:

Agrega un espacio entre elemntos y entre el contenedor y los elementos del mismo tamaño

Aling-items

Alínea los elemntos en el eje Y.

  • Flex-start:

Valor por default. Alínea los elementos en top.

  • Flex-end:

Alínea a los elementos en la parte de abajo.

  • Center:

Establece un valor de aling-self sobre todos los desendientes directos de un grupo. Alínea los elementos en el centrod del eje Y.

  • Strech:

Los lementos ocupan todo el alto de nuestro contenedor

  • Baseline:

Alínea los elementos de tal forma que se comprimen al tamaño del texto y se disponen teniendo en cuenta la base de la línea inferior del texto. Esto provoca que todos los elementos (así sean de distintos tamaños), se posiciónen en funcion de la misma medida inicial, creando un orden, dentro del "caos".
